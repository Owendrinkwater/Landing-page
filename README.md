# Landing Page Project

This project focuses on building a complete landing page using HTML and CSS,
with an emphasis on layout and flexbox.

## 🚀 Overview

The goal of this project is to recreate a landing page design
from a provided image. It tests the ability to structure content,
apply styling, and use flexbox for layout and alignment.

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox

## 📚 What I Learned

- Structuring a webpage using semantic HTML
- Using flexbox to create responsive layouts
- Aligning and spacing elements properly
- Styling text, buttons, and sections consistently
- Managing CSS by organizing selectors and avoiding duplication

## 📁 Project Structure

landing-page/

├── index.html
├── style.css
└── images/

## 🎯 Features

- Header with navigation links
- Hero section with call-to-action
- Information section with cards
- Quote/testimonial section
- Footer

## 💡 Challenges Faced

- Understanding flexbox alignment and positioning
- Managing spacing using margin and padding
- Avoiding repetition in CSS (e.g., duplicate selectors)

## ✅ Improvements

- Add responsiveness for mobile devices
- Improve visual design and spacing
- Refactor CSS for better scalability
